unsafe fn set_head(&mut self, index: u64)

Set the earliest position retained in the stream.

Panics

May panic if the provided an index earlier than the current head.

Safety

Only set the head if you have unique ownership of this and will not attempt to read any data earlier than index.

[src]

unsafe fn compare_set_head(&self, index: u64)

Set the earliest position retained in the stream.

This function, unlike set_head, is synchronized between threads. If the provided head is less than the current head, the current head remains unchanged.

Safety

See set_head.

    cx: &mut Context<'_>,
    register_wakeup: impl FnOnce(&Waker),
    index: u64,
    len: usize
) -> Poll<Result<usize, Self::Error>>

Suspends the current task until len samples starting at index are available, returning the available length.

If the stream is closed, the returned available length may be less than len.
