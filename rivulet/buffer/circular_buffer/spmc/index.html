rivulet::buffer::circular_buffer

[−][src]Module rivulet::buffer::circular_buffer::spmc

A single-producer, multiple-consumer async circular buffer.

This buffer has readers that implement Clone, at the cost of locking. For a lock-free buffer see spsc.
